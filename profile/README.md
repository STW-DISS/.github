# Distributed Intelligent Software Systems (DISS)

Welcome to the public code organization of the STW Distributed Intelligent Software Systems (DISS). The DISS GitHub organization serves as the central workspace for research prototypes, experimental software, and technical explorations conducted within the DISS transfer center. This space brings together engineering concepts, research activities, and hands-on experimentation in domains where distributed systems and intelligent software meet.

Our goal is to maintain a structured, transparent, and collaborative environment for code, experiments, and project materials produced in the context of academic-industry research.
